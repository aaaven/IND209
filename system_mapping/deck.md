## A four-step method

---

### Step 1 Start with One AI Interaction

A small, recognizable action pattern that a user performs with AI, especially one that appears habitual or repetitive, convenience-driven, excessive, or disproportionate relative to its infrastructural cost and planetary consequences.

---

#### 1.1 Identify the AI Type

What kind of AI system is involved?

For example:

<ul>
<li class="fragment">LLM / chatbot</li>
<li class="fragment">image generation model</li>
<li class="fragment">recommender system</li>
<li class="fragment">translation model</li>
<li class="fragment">AI search</li>
<li class="fragment">voice assistant</li>
<li class="fragment">productivity software with AI features</li>
</ul>

---

#### 1.2 Identify the AI Behavior

What exactly is the user doing? For example:

<ul>
<li class="fragment">saying “thank you” to ChatGPT</li>
<li class="fragment">asking ChatGPT to slightly reword one sentence five times</li>
<li class="fragment">asking AI to rewrite a message that was already clear</li>
<li class="fragment">using AI to summarize an already short, e.g. two-line, email</li>
<li class="fragment">generating ten versions of an image just to choose one</li>
<li class="fragment">using AI for tiny convenience tasks instead of human judgment</li>
</ul>

---

#### 1.3 Describe the Behavioral Pattern

Is it:

<ul>
<li class="fragment">one-off</li>
<li class="fragment">repeated</li>
<li class="fragment">iterative</li>
<li class="fragment">compulsive</li>
<li class="fragment">convenience-driven</li>
<li class="fragment">exploratory</li>
<li class="fragment">avoidant</li>
<li class="fragment">excessive</li>
</ul>

This is important because the assignment is about patterns of usage.

---

#### 1.4 Ask Whether the Use Is Proportionate

<ul>
<li class="fragment">Is this use necessary?</li>
<li class="fragment">Is the output worth the compute?</li>
<li class="fragment">Is the behavior replacing judgment, effort, or attention?</li>
<li class="fragment">Is this efficient, or is it a kind of convenience excess?</li>
<li class="fragment">Is the system doing more work than the task deserves?</li>
</ul>

This is where the critical layer comes in: disproportionate AI usage.

---

Trace what's behind the interaction
